Sum of Two Numbers (Easy)

Given two integers, return their sum. Write a function that takes two inputs and returns their addition.

Find Missing Number (Medium)

Given an array of integers from 1 to n with one number missing, find the missing number. For example, for the input [1, 2, 4, 5], the output should be 3.

Shortest Path in Graph (Hard)

Given a graph and a source node, find the shortest path to all other nodes using Dijkstra's algorithm. Assume the graph is represented as an adjacency matrix.

Reverse a String (Easy)

Write a function that takes a string as input and returns the string reversed.

Check for Palindrome (Easy)

Given a string, determine if it is a palindrome (reads the same backward as forward).

Find Maximum Element (Easy)

Given an array, find and return the maximum element in the array.

Find Even Numbers in an Array (Easy)

Given an array of integers, return an array containing only the even numbers.

Find Prime Numbers (Medium)

Write a function to find all prime numbers up to a given number n.

Find First Non-Repeated Character (Medium)

Given a string, find the first non-repeated character in it.

Sort an Array (Medium)

Implement a function that sorts an array in ascending order using any sorting algorithm.

Find Longest Substring Without Repeating Characters (Medium)

Given a string, find the length of the longest substring without repeating characters.

Find Strongly Connected Components (Hard)

Implement an algorithm to find strongly connected components in a directed graph.

Maximum Flow in a Network (Hard)

Solve the maximum flow problem in a network using the Ford-Fulkerson algorithm.

Traveling Salesman Problem (Hard)

Implement a solution for the traveling salesman problem using dynamic programming.

Knapsack Problem (Hard)

Solve the 0/1 knapsack problem using dynamic programming to find the maximum value of items to be placed in a knapsack.
